# Changelog

## [1.0.0] - 2026-07-16

### Added

- Public open-source release of the classroom geometry sketch board.
- Visible categorized command layout across top and side rails.
- Middle-school and high-school focused geometry tools, function graphs, and construction helpers.
- Coordinate snapping, integer-axis snapping, and exact coordinate point entry.
- AI-assisted drawing, step-by-step solving, equation display, and intersection display.
- iPad-friendly control sizing and macOS/Windows shortcut support.
- Public README, contribution guide, MIT license, syntax check script, and regression test suite.
