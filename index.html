操作说明 / CAD 快捷键

基础操作

点击 / Pencil点选对象；绘图时指定点位，iPad 支持 Apple Pencil。
Shift+点击多选或从选择中移除对象。
拖拽空白框选完整落入框内的对象。
中键/Alt拖拽平移画布。
滚轮双指/滚轮平移；Ctrl+滚轮缩放。
右键非绘制状态打开菜单；绘制中确认/结束或撤销当前点。

确认 / 取消

Space完成当前折线/多边形；空闲时重复上一个绘图命令。
Backspace绘制中撤销上一个点；选中对象时删除。
Esc取消当前命令；清空选择并回到选择工具。
Delete删除选中对象。
Cmd/Ctrl+Z撤销；Cmd/Ctrl+Shift+Z 或 Ctrl+Y 重做。
方向键平移视图；Shift+方向键快速平移。

绘图命令

L线段。
XL / X直线。
R / REC矩形。
C圆。
A弧线。
P / PL折线。
PG多边形。
RT精确直角三角形。
QF二次函数 y=ax²+bx+c。
T文字。
D尺寸标注。
AN角度标注。
角标控件选择 1弧/2弧/3弧表示不同相等角；选直角可画方角。
AX坐标系。
F区域填充。

编辑命令

CO / CP复制选中对象。
Cmd/Ctrl+C/V复制 / 粘贴选中对象。
[ / ]逆时针 / 顺时针旋转选中对象 15°。
E删除选中对象。
M点对点移动：选中对象→指定基点→指定目标点，支持吸附。
MH围绕对象中心水平镜像。
MV围绕对象中心垂直镜像。
CMH / CMV保留原对象，生成水平/垂直镜像副本。
MI轴线镜像：选中对象后，点击镜像轴两点。
角度工具点第一条边→点第二条边→向外拉开确定角标位置；选中后可调半径。
Z缩放到全图。
双击文字/标注编辑文字内容或标注文本。
选中标注+Enter快速编辑尺寸/角度/文字标注；尺寸标注可在右侧调整偏移。
选中后控制点拖蓝色方块改顶点/端点；拖绿色圆点改尺寸偏移或角标半径。
